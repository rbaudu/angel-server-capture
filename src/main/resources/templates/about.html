À propos d'Angel Server

Serveur de capture, synchronisation et analyse de flux vidéo et audio.

Présentation

Angel Server est une application dédiée à la capture, synchronisation et analyse de flux vidéo et audio en temps réel. Développée en Java avec Spring Boot, elle utilise les bibliothèques OpenCV et JavaCV pour le traitement vidéo.

Cette application permet :

  • La capture vidéo depuis des caméras connectées
  • La capture audio depuis des microphones
  • La synchronisation précise des flux vidéo et audio
  • L'analyse en temps réel des flux (détection de mouvement, de personnes, etc.)
  • La visualisation en direct des flux capturés

L'architecture modulaire d'Angel Server facilite son extension avec de nouvelles fonctionnalités d'analyse ou de traitement.

Technologies utilisées

Backend

  • Java 17
  • Spring Boot 3.2
  • Spring WebSocket
  • JavaCV / OpenCV
  • FFmpeg (pour le traitement audio)

Frontend

  • Thymeleaf
  • Bootstrap 5
  • JavaScript / jQuery
  • SockJS / STOMP
  • Font Awesome
